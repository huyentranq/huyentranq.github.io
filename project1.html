TMDB-Pipeline-Recommendation

Dự án ELT Pipeline thuộc lĩnh vực Data Engineering nhằm hỗ trợ hệ thống gợi ý phim và dashboard phân tích.

🎯 Mục tiêu dự án

  • Xây dựng hệ thống pipeline hoàn chỉnh từ thu thập đến trực quan hóa dữ liệu
  • Gợi ý phim theo lịch sử đánh giá cá nhân
  • Phân tích và báo cáo dữ liệu phim qua dashboard

🚀 Công nghệ sử dụng

Orchestration & Data Processing

Python, Apache Spark, Dagster, DBT

Lưu trữ & Truy xuất

MinIO, MySQL, PostgreSQL, Polars

Visualization

Streamlit

🔧 Kiến trúc Pipeline

  • Data Sources: TMDB API, Kaggle Dataset (1M movies)
  • Bronze: dữ liệu thô lưu tại MySQL
  • Silver: xử lý và làm sạch với Spark/Polars
  • Gold: enrich và tổ chức dữ liệu
  • Warehouse: PostgreSQL và DBT
  • UI: Streamlit Recommendation + Dashboard

🧬 Data Lineage

Quản lý job và assets bằng Dagster

⚙️ Các bước triển khai

  1. Clone repo & tải dataset từ Kaggle
  2. Điền file ENV với API Token từ TMDB
  3. Thiết lập môi trường ảo Python
  4. Build container bằng Makefile
  5. Load dữ liệu vào MySQL và PostgreSQL
  6. Chạy các assets qua Dagster
  7. Build models bằng DBT
  8. Khám phá ứng dụng tại Streamlit

📸 Giao diện Streamlit

🎯 Kết luận

Dự án giúp mình tiếp cận với toàn bộ quy trình ELT hiện đại, từ ingest, transform đến build data warehouse và UI. Đây là nền tảng vững chắc để phát triển các hệ thống dữ liệu phức tạp trong tương lai.
